# Wedding Gallery Web

A premium wedding gallery web app built with React, TypeScript, Vite, Tailwind CSS, Motion, React Router, and Zustand.

## Scripts

- `npm run dev`
- `npm run build`
- `npm test`
- `npm run preview`

## Environment

- Node `22.22.1`

The app now includes:

- editorial home and event experiences
- mobile-adaptive navigation and sheets
- premium single-photo viewer
- albums, saved photos, and people flows
- shared feedback system and route-transition polish
- initial automated regression coverage
